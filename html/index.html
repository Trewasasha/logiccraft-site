Актуальность и проблема

Сегодня существует огромный разрыв между проектированием и разработкой. Архитектор рисует красивую диаграмму, а программист затем вручную пишет десятки классов — поля, методы, связи. Диаграмма остается лишь в документации и мгновенно устаревает, а рутина отнимает часы работы и ведет к архитектурным ошибкам.

UI/UX Architecture Под капотом

Интеллектуальная панель свойств

Правая панель свойств строго разделена на эргономичные зоны: имя, тип, атрибуты, методы и кнопка удаления. При клике на любую карточку класса на холсте панель мгновенно перестраивается и отображает её актуальные данные.

Внутри это изолированный класс PropertiesPanel на базе слоя QVBoxLayout, оперирующий компонентами QLineEdit, QComboBox и QListWidget. Панель не хранит состояние внутри себя — она выступает чистым View и только транслирует сигналы изменений в центральный контроллер.

Наша Фишечка — Debounce

QTimer на 500 миллисекунд

Панель свойств не дергает движок на каждый ввод символа. Она дожидается логической паузы пользователя в 500 мс с помощью механизма Debounce.

✓ Снижает вычислительную нагрузку на GUI поток
✓ Исключает мусорные состояния из истории Undo/Redo

class PropertiesPanel(QWidget):
    # Сигналы для отправки изменений в контроллер
    name_changed = pyqtSignal(str, str)        # card_id, new_name
    attributes_changed = pyqtSignal(str, list) # card_id, attributes
    methods_changed = pyqtSignal(str, list)    # card_id, methods

    def __init__(self, parent=None):
        super().__init__(parent)

        # Поля для редактирования
        self.name_edit = QLineEdit()
        self.attrs_list = QListWidget()
        self.methods_list = QListWidget()

        # Таймер для задержки (Debounce)
        self._name_timer = QTimer()
        self._name_timer.setSingleShot(True)
        self._name_timer.timeout.connect(self._emit_name_changed)

        # Подключаем сигналы
        self.name_edit.textChanged.connect(self._on_name_changed)

    def _on_name_changed(self, text):
        """При вводе текста перезапускаем таймер"""
        self._name_timer.start(500)  # Ждём 500 мс

    def _emit_name_changed(self):
        """Отправляем изменение только после паузы пользователя"""
        self.name_changed.emit(self.current_card_id, self.name_edit.text())

    def load_card(self, card_id, name, attributes, methods):
        """Загружаем данные выбранной карточки"""
        self.current_card_id = card_id
        self.name_edit.setText(name)
        self.attrs_list.clear()
        self.attrs_list.addItems(attributes)
        self.methods_list.clear()
        self.methods_list.addItems(methods)

def push_state(self, state: Union[UMLDiagram, dict]) -> None:
    """Добавляет новое состояние в историю (с фильтрацией микрокликов)"""
    if self._is_restoring:
        return

    if isinstance(state, dict):
        try:
            state = UMLDiagram.model_validate(state)
        except Exception as e:
            logger.error(f"History: Validation error: {e}")
            return

    # 1. Проверка на дубликат или микро-сдвиг (Deadzone)
    is_dup, reason = self._check_duplicate(state)
    if is_dup:
        return

    # 2. Создание глубокой копии вне лока (Оптимизация FPS)
    new_snapshot = state.model_copy(deep=True)

    # 3. Атомарная потокобезопасная запись
    with self._lock:
        if self._current_index < len(self._stack) - 1:
            self._stack = self._stack[:self._current_index + 1]

        self._stack.append(new_snapshot)
        self._current_index += 1

        if len(self._stack) > self._max_history:
            self._stack.pop(0)
            self._current_index -= 1

    self._emit_status()

// Построение карты наследования для генерации корректной структуры ООП
def _build_inheritance_map(self, diagram: UMLDiagram) -> Dict[str, List[str]]:
    inheritance_map = {}
    for node in diagram.nodes:
        inheritance_map[node.name] = []

    if hasattr(diagram, 'connections'):
        for connection in diagram.connections:
            if connection.type.value == "inheritance":
                source_node = next((n for n in diagram.nodes if n.id == connection.source_id), None)
                target_node = next((n for n in diagram.nodes if n.id == connection.target_id), None)
                if source_node and target_node:
                    inheritance_map[source_node.name].append(target_node.name)

    return inheritance_map
